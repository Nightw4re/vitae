# Vitae Wiki

Vitae is a data-driven framework for custom Minecraft entities on NeoForge 1.21.1.

Use this wiki as the project reference for:
- entity JSON schema
- ability composition
- boss phases and transitions
- demo content
- spawn rules
- development workflow
